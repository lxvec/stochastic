Total Variation & Quadratic Variation

MATH 60646A HEC Montréal  ·  Smooth f(t) vs. Brownian W(t) as the partition n → ∞

Step-by-step:  f(t) = t  with 10 partition intervals
Paths with n = 10 partition intervals  ·  segments colored by increment magnitude
Absolute increments |Δf_i| — bars show each term in the TV sum
Convergence of TV_n and QV_n as partition size n increases

For a partition π_n = {t_0=0 < t_1 < ⋯ < t_n=1}, the n-th order total variation is:

TV_n(f)  =  Σᵢ₌₀ⁿ⁻¹ |f(t_{i+1}) − f(t_i)|
Smooth functions
TV_n(f) → ∫₀¹ |f'(t)| dt < ∞ as n → ∞. Each term |Δf_i| ≈ |f'(t_i)|·(1/n), so n terms of size 1/n → constant sum. The path has bounded variation.
Brownian motion
TV_n(W) → +∞ a.s. Each increment |ΔW_i| ≈ √(1/n) in magnitude, so n terms of size 1/√n → TV_n ~ √n → ∞. BM has unbounded variation — the red curve diverges!

Unbounded variation of BM means the classical Riemann–Stieltjes integral ∫ f dW is not well-defined path-by-path. This is the core reason why Itô's construction is needed.

The n-th order quadratic variation is:

QV_n(f)  =  Σᵢ₌₀ⁿ⁻¹ (f(t_{i+1}) − f(t_i))²

The quadratic variation process [f]_t is the limit as the mesh → 0:

[f]_T  =  lim_{n→∞} QV_n(f)
Smooth functions: [f]_T = 0
Each (Δf_i)² ≈ (1/n)², so n terms of size (1/n)² → sum ~ 1/n → 0. Switch to QV mode and watch the blue bars all shrink rapidly to zero as n grows.
Brownian motion: [W]_T = T = 1
Each (ΔW_i)² ≈ 1/n, so n terms each ~ 1/n → sum → 1 = T. Watch the orange QV_n(BM) line in the convergence chart stabilize near 1!

Intuition: BM increments scale as √Δt, not Δt. Squaring gives (√Δt)² = Δt, and summing n terms each of size Δt = 1/n gives T = 1. The squares don't vanish fast enough.

In ordinary calculus, the second-order Taylor term vanishes because (dx)² → 0 for smooth paths:

f(x_T) = f(x_0) + ∫₀ᵀ f'(x_t) dx_t   [classical chain rule]

For Brownian motion, (dW_t)² = dt ≠ 0, so the second-order term survives. Expanding f(W) via Taylor and collecting the surviving (dW)² term gives:

f(W_T) = f(W_0) + ∫₀ᵀ f'(W_t) dW_t + ½ ∫₀ᵀ f''(W_t) dt   [Itô's formula]

The green term is the Itô correction — it is exactly ½ times the quadratic variation density. Since [W]_t = t (so d[W]_t = dt), this extra term is unavoidable.

Example: For f(x) = x², Itô gives: W_T² = 0 + ∫₀ᵀ 2W_t dW_t + ½·2·T, so W_T² = 2∫W_t dW_t + T. Classical calculus (wrong): W_T² = 2∫W_t dW_t — it's off by T!

Quick Reference

TV_n(f):
Σ |Δf_i|  →  TV(smooth) < ∞
Σ |ΔW_i|  →  +∞ a.s., grows ~√n

QV_n(f):
Σ (Δf_i)²  →  0 for smooth f
Σ (ΔW_i)²  →  T = 1 a.s.

Key identity:
(dW_t)² = dt  ⟺  [W]_t = t

Itô's formula:
df(W_t) = f'(W_t)dW_t + ½f''(W_t)dt
↑ extra term from [W]_t = t ≠ 0

Exercise 8.1 — Ornstein–Uhlenbeck SDE

MATH 60646A HEC Montréal  ·  dXt = κ(θ − Xt) dt + σ dWt  ·  Drift & diffusion decomposition with Monte Carlo simulation

Chart 1 — Drift coefficient  b(Xt) = κ(θ − Xt)  as a function of Xt κ > 0 — mean-reverting
Chart 2 — Diffusion contribution   σ·W(t)   (K independent paths)
Diffusion contribution alone: σ·dWt. Each path is i.i.d.; spread grows like ±σ√t regardless of Xt (constant diffusion coefficient).
Chart 3 — Combined:  dXt = κ(θ − Xt)dt + σ dWt   (Euler–Maruyama)

dX_t = 1·(1 − X_t)dt + 1·dW_t

Since X₀ = θ = 1 the process starts exactly at its long-run mean. The drift is zero at t=0 and the evolution is driven entirely by noise. The process is mean-reverting (κ=1 > 0): if noise pushes X_t above 1 the drift pulls it back down; if below 1 the drift pushes it back up. Over time X_t ≈ N(1, σ²/(2κ)) = N(1, 0.5) in the stationary distribution.

  • Long-run mean E[X_∞] = θ = 1
  • Long-run variance Var[X_∞] = σ²/(2κ) = 0.5
  • Half-life of deviations = ln(2)/κ ≈ 0.693 time units

dX_t = −1·(1 − X_t)dt + 1·dW_t = (X_t − 1)dt + dW_t

With κ = −1 < 0 the drift now repels X_t away from θ=1. Any small perturbation from 1 is amplified: if X_t > 1, the drift is positive (pushing it further up); if X_t < 1, the drift is negative (pushing it further down).

In Chart 1 you will see the drift line has a positive slope (slope = −κ = 1), so the force arrow at X_t > 1 points right, and at X_t < 1 points left — always away from θ.

The process is explosive: E[X_t] = 1 + (X₀−1)e^t = 1 (starts exactly at 1), but any deviation grows exponentially. Paths diverge to ±∞.

dX_t = 1·(1 − X_t)dt + 0.1·dW_t

Same drift as case (a) but the noise σ=0.1 is ten times smaller. The paths stay very close to the deterministic ODE solution X_t^{det} = 1 (since X₀=θ=1, the ODE solution is flat at 1). In Chart 2 the diffusion paths are very tightly clustered around 0.

Stationary variance = σ²/(2κ) = 0.005 — extremely concentrated near θ=1. Chart 3 illustrates how reducing σ brings the stochastic paths closer to the ODE reference line.

dX_t = 1·(2 − X_t)dt + 0.3·dW_t

Here X₀=1 < θ=2, so at t=0 the drift is κ(θ−X₀) = 1·(2−1) = +1 > 0. The drift is pulling X_t upward toward 2. Chart 1 clearly shows the force arrow at X_t=1 pointing rightward (positive drift).

The ODE solution is X_t^{det} = 2 + (1−2)e^{−t} = 2 − e^{−t}, starting at 1 and converging to 2. The stochastic paths track this deterministic path with fluctuations of size σ/√(2κ) = 0.212.

Stationary distribution: N(2, 0.045). The convergence takes ~5 half-lives = 5·ln2 ≈ 3.5 time units to get within 3% of θ.

Exact solution (via integrating factor e^{κt}):

X_t = θ + (X₀ − θ)e^{−κt} + σ ∫₀ᵗ e^{−κ(t−s)} dW_s
Moments (κ > 0)
E[X_t] = θ + (X₀−θ)e^{−κt}
Var[X_t] = (σ²/2κ)(1 − e^{−2κt})
E[X_∞] = θ,  Var[X_∞] = σ²/(2κ)
Half-life = ln(2)/κ
Moments (κ < 0)
E[X_t] = θ + (X₀−θ)e^{−κt} → ±∞
Var[X_t] = (σ²/2κ)(1−e^{−2κt}) → +∞
No stationary distribution.
Paths explode exponentially.

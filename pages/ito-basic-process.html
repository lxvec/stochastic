Itô Integral — Basic Process (Lemma 1)

Slide 19/79 · Gauthier, Dec 2024 · MATH 60646A HEC Montréal

Path of a basic process and its stochastic integral w.r.t. Brownian motion
Click anywhere on the chart to pin coordinates for comparison · click a pinned marker again to unpin
Distribution of I(1) = C·(W(b) − W(a)) across simulations

Regime 1: t ≤ a
X(t) = 0, so no integrand is active. I(t) = 0 — the integral has not started accumulating. The red line sits flat at zero.
Regime 2: a < t ≤ b
X(t) = C (active). I(t) = C·(W(t) − W(a)). The red line tracks the Brownian path, scaled by C and re-anchored at W(a).
Regime 3: t > b
X(t) = 0 again. I(t) = C·(W(b) − W(a)) — frozen. The red line goes perfectly horizontal. The integral "knows" nothing after b.

A process M(t) is a martingale if E[M(t) | Fs] = M(s) for s ≤ t. For I(t) = C·(W(t∧b) − W(a)):

  • E[I(T)] = 0 always, because C is Fa-measurable and W(b) − W(a) is independent of Fa with mean 0.
  • Var[I(T)] = C²·(b − a) — the Itô isometry ‖∫ X dW‖² = ‖X‖²L² = E[∫ X² dt] = C²·(b−a).
  • After time b, I(t) is constant → its conditional expectation given any later information is just itself. No drift after the interval closes.

Quick Reference

Definition (basic process):
X(t) = C · 1(a,b](t), where C is Fa-measurable

Stochastic integral:
I(t) = ∫₀ᵗ X(s) dW(s) = C · (W(t∧b) − W(a))

Isometry:
E[I(1)²] = E[∫₀¹ X(t)² dt] = C²·(b−a)

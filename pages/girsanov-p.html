Market price of risk — predictable process (CIR)
For the short rate rt (here the simulated CIR state Xt), the predictable market price of risk is
γt = (κℚ − κℙ)√rt

Girsanov (same notation as course notes): dWtℚ = dWtℙ + γt dt  ⇒  W̃t := Wtℙ + ∫0t γs ds is a ℚ–Brownian motion (same ω; integral on the EM grid along simulated X).
Chart: the pink Wℚ trace is a separate standard Brownian path (own seed), same law as ℚ–Wiener but not tied to σ or the CIR path — so it stays fixed when you move σ (unlike γ or W̃).

Assumption (imposed on this page): κℙθℙ = κℚθℚ. Then κℙθℙ − κℚθℚ = 0, so the second summand in γt above vanishes and γt = (κℚ − κℙ)√rt/σ. The chart’s γ trace uses this reduced expression (the second summand above is treated as zero under the assumption).

Quick Reference

CIR SDE under ℙ
dXt = κ(θ−Xt)dt + σ√Xt dWt

Wt — pure Brownian motion under ℙ; starts at 0, drives the whole process via dWt

Drift only — simulate X using only κ(θ−Xt)dt; deterministic ODE, exponential convergence toward θ

Diffusion only — simulate X using only σ√Xt dWt; random noise scaled by √X, no pull toward θ

σ·Wt — appears with diffusion toggle; the displayed BM multiplied by σ, showing the raw scaled noise before the √Xt state-dependent weighting

Full CIR — appears when all three toggles are on; the combination of drift + diffusion

γt — CIR predictable market price of risk (see top panel); plotted on the right axis along the simulated full CIR path Xt = rt.

Wtℚ — same ω as Wℙ, with Wtℚ = Wtℙ + ∫0tγsds from the γ formula and κℚ, θℚ sliders.

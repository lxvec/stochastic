Conditional Expectation E[X | F]

The conditional expectation E[X | F] is the best prediction of X given the information in σ-algebra F. It is constant on atoms of F.

Property: E[X | F] is F-measurable (constant on F-atoms)

Mathematical Definition

Conditional Expectation: Given a random variable X and σ-algebra F ⊆ Σ,

E[X | F] is the unique F-measurable random variable such that:

  • For each atom A ∈ F: E[X | F] is constant on A
  • E[X | F](ω) = (1/|A|) Σω'∈A X(ω') for ω ∈ A
  • This is the average value of X over the atom containing ω

Intuition: E[X | F] represents the best prediction of X when you only know which atom of F the true state ω belongs to.

Example

Suppose X(A₁) = 100, X(A₂) = 150, X(A₃) = 120, X(A₄) = 180

E[X] = (100 + 150 + 120 + 180) / 4 = 137.5

If F = σ({A₁, A₂}, {A₃, A₄}):

  • For ω ∈ {A₁, A₂}: E[X | F] = (100 + 150) / 2 = 125
  • For ω ∈ {A₃, A₄}: E[X | F] = (120 + 180) / 2 = 150
